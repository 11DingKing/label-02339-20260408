# 环境监测系统 (EnvMonitor)

基于 Qt 6 的环境参数实时监测与管理系统，集成串口通信、TCP/UDP 网络上传、SQLite 数据存储。

## 快速开始

### 一键启动（推荐）

**macOS / Linux:**
```bash
chmod +x run.sh
./run.sh
```

**Windows:**
```cmd
run.bat
```

脚本会自动：
- 检测并安装缺失的依赖（Qt6、CMake）
- 编译项目
- 初始化数据库并导入测试数据（89条）
- 启动测试服务器（端口 9000，用于网络上传测试）
- 启动程序

### 快速体验

启动后点击工具栏「一键模拟」按钮：
- 自动启动串口模拟，生成环境数据
- 自动连接测试服务器（127.0.0.1:9000）
- 自动上传数据到服务器
- 数据自动存入数据库

无需任何配置即可体验全部功能。

### 手动编译运行

**前置条件：**
- Qt 6.x（含 SerialPort、Network、Sql 模块）
- CMake 3.16+
- C++17 编译器

**步骤：**

```bash
cd backend
mkdir -p build && cd build
cmake .. -DCMAKE_PREFIX_PATH=/path/to/your/Qt6
make -j$(nproc)

# 初始化数据库（可选）
sqlite3 ./envmonitor.db < ../schema.sql
sqlite3 ./envmonitor.db < ../init_data.sql

./EnvMonitor
```

> macOS Homebrew 用户：`cmake .. -DCMAKE_PREFIX_PATH=$(brew --prefix qt)`

### 技术要点

- **一键模拟**：同时启动串口模拟和网络上传，方便快速测试
- **采集周期**：仅对模拟模式生效，真实串口由外设主动上报
- **自动重连**：TCP断连后指数退避重连（3s→6s→12s...最大60s），无次数上限
- **网络模块**：客户端上传模式，非服务端
- **Toast提示**：所有按钮操作都有即时反馈
- **中文界面**：所有弹窗按钮为中文（确定/取消）
- **日志文件**：关键错误写入 `logs/envmonitor_YYYYMMDD.log`
